QEMU GPU Passthrough Methodology

The following are the steps that I take in order to set up a Windows, specifically 11, virtual machine under Pop!_OS Linux. If I am working with a new install of Pop!_OS, I run this script from Pavol Elsig. It takes all the grunt work out of hiding your GPU from the host os. I then follow this guide from Chris Titus. It walks you through installing the necessary packages, as well as some tweaks for networking and allowing your $USER to access the virtualization packages.
